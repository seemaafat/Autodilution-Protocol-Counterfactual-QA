# Autodilution Protocol Counterfactual QA

From-scratch Eris challenge. Solvers answer multiple-choice counterfactual questions by interpreting a liquid-handling DSL twice: original protocol and mutated protocol.

Upload `autodilution-counterfactual-qa-raw.zip` as the raw dataset. Use `problem.md`, `grade.py`, `prepare.py`, `config.yaml`, and `rubrics.yaml` in the challenge editor.
